## --- HOW TO RUN ---

Place the exe in the same directory as the endfield.exe
then run the unlocker exe and input the desired FPS value.

## -- HOW TO UNINSTALL
Remove vulkan-1.dll from root directory
Remove ace_inject.dll from root directory
